  • set的用处：有时候我们不想要key与value的对应关系，只想要dict的key，不关心value，目的只是让集合不要重复。它无序且不重复。
    s = set(['A', 'B', 'C'])
    查看set的内容：
    print s
    set(['A', 'C', 'B'])
    当出现重复元素时，会自动滤去。
    由于set存储的是无序集合，所以我们没法通过索引来访问。

  • set的特点：set的内部结构和dict很像，1）唯一区别是不存储value。set存储的元素和dict的key类似，2）必须是不变对象。set存储的元素也是3）没有顺序的。

  • def定义函数：依次写出函数名、括号、括号中的参数和冒号，然后，在缩进块中编写函数体，函数的返回值用 return 语句返回。注：如果没有return语句，函数执行完毕后也会返回结果，只是结果为 None，return None可以简写为return。
    实例：请定义一个 square_of_sum 函数，它接受一个list，返回list中每个元素平方的和。

  • 函数返回多值：实质是返回一个tuple。
    实例：一元二次方程的定义是：ax² + bx + c = 0，请编写一个函数，返回一元二次方程的两个解。

  • 递归函数：一个函数在函数体内调用自身，理论上，所有的递归函数都可以写成循环的方式，但循环的逻辑不如递归清晰。使用递归函数需要注意防止栈溢出。在计算机中，函数调用是通过栈（stack）这种数据结构实现的，每当进入一个函数调用，栈就会加一层栈帧，每当函数返回，栈就会减一层栈帧。由于栈的大小不是无限的，所以，递归调用的次数过多，会导致栈溢出。
    实例：汉诺塔 (http://baike.baidu.com/view/191666.htm) 的移动也可以看做是递归函数。我们对柱子编号为a, b, c，将所有圆盘从a移到c可以描述为：如果a只有一个圆盘，可以直接移动到c；如果a有N个圆盘，可以看成a有1个圆盘（底盘） + (N-1)个圆盘，首先需要把 (N-1) 个圆盘移动到 b，然后，将 a的最后一个圆盘移动到c，再将b的(N-1)个圆盘移动到c。请编写一个函数，给定输入 n, a, b, c，打印出移动的步骤：move(n, a, b, c)例如，输入 move(2, ‘A’, ‘B’, ‘C’)，打印出：A –> B A –> C B –> C

  • 默认参数：例如Python自带的 int() 函数，其实就有两个参数，我们既可以传一个参数，又可以传两个参数：第二个参数是转换进制，如果不传，默认是十进制 (base=10)，如果传了，就用传入的参数。函数的默认参数的作用是简化调用，但是在需要的时候，又可以传入额外的参数来覆盖默认参数值。由于函数的参数按从左到右的顺序匹配，所以默认参数只能定义在必需参数的后面。（默认参数，传入是就修改默认值，未传入时，就使用原来的值，类似：(a, b=1, c=2)）
    实例：请定义一个 greet() 函数，它包含一个默认参数，如果没有传入，打印 ‘Hello, world.’，如果传入，打印 ‘Hello, xxx.’

  • 定义可变参数：可变参数的名字前面有个“星号”，我们可以传入0个、1个或多个参数给可变参数。Python解释器会把传入的一组参数组装成一个tuple传递给可变参数，因此，在函数内部，直接把变量 args 看成一个 tuple 就好了。
    实例：请编写接受可变参数的 average() 函数。
