Linux入门笔记（一）_补充

发表于 2018-01-08 | 分类于 Linux
字数统计: 1,059 字 | 阅读时长 ≈ 4 分钟
  • 进入上一级目录：cd ..
  • 进入你的home目录：cd ~ （cd/home/<你的用户名>）
  • 使用pwd可以获取当前路径
  • 绝对路径以/开头，相对路径以..开头
  • 新建目录：
    1）mkdir命令可以新建一个空目录，也可同时指定权限属性$ mkdir mydir
    2）使用-p参数同时创建父目录：$ mkdir -p father/son/grandson
  • 用cp命令复制文件到指定目录，如果要复制目录则加上-r或-R参数表示递归复制
  • rm进行删除操作，加上-f参数表示强制删除
  • mv命令是剪切移动文件，也可用于重命名单个文件
  • 批量重命名使用rename命令，举例：

rename 是先使用第二个参数的通配符匹配所有后缀为 .txt 的文件，然后使用第一个参数提供的正则表达式将匹配的这些文件的 .txt 后缀替换为 .c，这一点在学习了 sed 命令后会更好地理解。

  • 查看文件：cat 为正序显示，tac 为倒序显示。加上-n参数显示行号。
    nl 命令，添加行号并打印，这是个比 cat -n 更专业的行号打印命令。常用参数还有：

  • more和less命令用来阅读文件内容。

  • head和tail命令可以只看文件的头几行个尾几行（默认10），加上 -n 参数，后面紧跟行数。
  • file命令查看文件类型
  • 查看用户：who命令，常用：
    -a 打印能打印的全部
    -d 打印死掉的进程
    -m 同am i,mom likes
    -q 打印当前登录用户数及用户名
    -u 打印当前登录用户登录信息
    -r 打印运行等级
  • 用户组：groups命令，每次新建用户如果不指定用户组的话，默认会自动创建一个与用户名相同的用户组。
  • 删除用户：$ sudo deluser lilei --remove-home
  • 查看文件权限：ls -l,ll
  • 显示所有文件，包括隐藏文件（Linux 下以 . 开头的文件为隐藏文件）：-A
  • 查看一个目录的完整属性：$ ls -dl <目录名>
  • 显示所有文件大小：$ ls -AsSh
  • 常用快捷键：
    Tab 命令补全
    Ctrl+c 强制退出
    Ctrl+d 键盘输入结束或退出终端
    Ctrl+s 暂停当前程序，暂停后按下任意键恢复运行
    Ctrl+z 将当前程序放到后台运行，恢复到前台为命令fg
    Ctrl+a 将光标移至输入行头，相当于Home键
    Ctrl+e 将光标移至输入行末，相当于End键
    Ctrl+k 删除从光标所在位置到行末
    Alt+Backspace 向前删除一个单词
    Shift+PgUp 将终端显示向上滚动
    Shift+PgDn 将终端显示向下滚动
  • 通配符：星号和 ？
    当查找文件夹时，可以使用*来代替一个或多个真正字符；当不知道真正字符或者懒得输入完整名字时，常常使用通配符？代替一个或多个真正字符。
    其他通配符：
    [list] 匹配 list 中的任意单一字符
    [!list] 匹配 除list 中的任意单一字符以外的字符
    [c1-c2] 匹配 c1-c2 中的任意单一字符 如：[0-9] [a-z]
    {string1,string2,…} 匹配 string1 或 string2 (或更多)其一字符串
    {c1..c2} 匹配 c1-c2 中全部字符 如{1..10}
  • man命令用来获取帮助
