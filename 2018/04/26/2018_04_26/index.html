#字符串、向量和数组

  • using namespace：调用某个库函数时，只要用using namespace::name; 声明就可以了，可以一行放多个。头文件中包含的就不需要再有了。
  • 初始化：=是拷贝初始化，（）是直接初始化。初始值只有一个时都适用，多个初始值是直接初始化较好，拷贝的话需要显示的创建一个临时对象。（多个初始值比如string a=(10,’c’)是指cccccccccc由10和字符c两个参数创造）
  • string对象会自动忽略开头的空白（空格符、换行符、制表符等）知道下一个空白为止
  • getline：读取一整行，遇到换行符就停止并输出
  • if(!line.empty())限制非空，if(line.size()>80)只输出80个字符之后的，size函数返回的是string::size_type类型（是无符号的）的值，
  • 当string对象和字符字面值及字符串字面值时，确保+两侧至少有一个string
  • ispunct：判断是否为标点符号，isspace判断是否为空格
  • 标准库类型vector表示对象的集合，它是一个类模板。vector （个数，值）；vector {第一个，第二个，…}。{ }可用来进行列表初始化，但其中值必须与元素类型相同，如vector {10,”25”}表示10个值为“25”的元素但并不是初始化。
  • push_back可向vector中添加元素。
  • 迭代器：类似指针一样可以间接访问对象，使用了迭代器的循环体不可向其所属的容器中添加元素。注意end()函数指的是最后一个元素的后一个位置，而不是最后一个元素！！！
  • 数组：数组大小确定不变，而vector可以动态增加。其维度必须是一个常量表达式。定义数组时必须指明类型，不允许用类似auto的关键字。数组间不允许拷贝和赋值，不存在引用的数组。也不能用vector初始化数组，反之可以，vector之间也可以。
  • string *p = nums等价于*p = &nums[0]都是指向第一个元素
  • 如果指针指向的是一个元素，则允许解引用该结果指针。
  • C风格字符串：strcpy拷贝字符串，strcat将两个字符串连接，strcmp比较两个字符串。无法用string对象初始化C风格字符串，要用到c str()。c str()：生成一个const char*指针，指向以空字符终止的数组。

  • int int_arr[] = { 0,1,2,3,4,5 };
    vector< int > ivec(begin(int_arr), end(int_arr)); 是数组对vector的初始化

  • int intarr[] = { 0,1,2,3,4,5 };
    vector< int > ivec(int     arr + 1, int_ arr + 4); 拷贝了int_arr[1],int_arr[2],int_arr[3]三个元素，左闭右开？？？

  • for语句处理多维数组时，除了最内层循环，都要用引用，如：for(auto &row:ia)

  • size_t是unsigned int的别称
  • a->b等价于*(a).b
